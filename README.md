# Nova

Nova is an open-source AI runtime platform for building persistent, multimodal, agentic companions and autonomous systems.

## Vision

Nova is not a robot.

Nova Rover is the first application built on top of Nova.

The long-term goal is to create a platform that supports:

- AI Companions
- Enterprise Agents
- Factory Intelligence
- Creator Intelligence
- Developer Agents
- Autonomous Systems

## Core Principles

1. Platform First, Robot Second
2. Everything is a Plugin
3. Observe → Understand → Plan → Act → Reflect
4. Identity and Memory are First-Class Citizens
5. Model Agnostic
6. Hardware Agnostic
7. Human Approval for Code Changes
